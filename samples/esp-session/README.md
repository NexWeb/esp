ESP Session Sample
===

This sample shows how to use ESP session state storage.

Requirements
---
* [ESP](http://embedthis.com/downloads/esp/download.esp)

To run:
---
    esp

The server listens on port 4000. Browse to: 
 
     http://localhost:4000/
