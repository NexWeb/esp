Compatibility

ESP APIs may evolve between minor releases. The ESP documentation provides a Stability Classification for each API group that reflects their stability with respect to forward changes.

Migrating to ESP from Appweb

ESP 1.X is mostly compatible with ESP in Appweb 4.X. Code written for Appweb 4.X ESP should migrate to ESP with few changes. The primary exception is use of the esp-legacy-mvc framework is not support. Apps using the esp-legacy-mvc framework should migrate to use either esp-html-mvc or esp-angular-mvc.
