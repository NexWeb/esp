ESP Quick Start

  1. Download

    First download ESP to your system. You can download a binary package from http://embedthis.com/downloads/esp/download.esp or source code from https://github.com/embedthis/esp.

  2. Installation

    Install ESP on your system by following the instructions relevant to your system. See Building from Source if building from source code.

  3. Quick Tour

    Take the ESP Tour. This will give you a quick overview of ESP and how to use it.

  4. Read the doc

    There is a wealth of documentation to read. Try the Users Guide first.
