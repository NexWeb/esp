ESP Quick Start

  1. Download ESP

    First download and install ESP on your system.

    Download ESP

  2. Install ESP

    Install ESP on your system by following the instructions relevant to your system.

  3. Quick Tour

    Take the ESP Tour. This will give you a quick overview of ESP and how to use it.

  4. Install Essential Tools

    If you are creating ESP applications, you will need two sibling tools from Embedthis.

    • Pak manager to install and manage extension packages.
    • Expansive site manager for layouts, and web site and content tooling.

    Download Pak Download Expansive

  5. Read the doc

    There is a wealth of documentation to read. Try the User's Guide first.
