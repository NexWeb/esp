Production Deployment

For deployment in a production environment, it optimal to compile the entire application including all controllers, views and web pages into a single library. This is refered to as "combined" mode. The normal mode is called "discrete" mode. To do a combined compile, set the esp.combine property in esp.json to true, and then invoke esp compile. For example:

esp: { combine: true }
