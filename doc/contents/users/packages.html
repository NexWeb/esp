Extension Packages

ESP can be extended by installable packages to provide content and extended functionality. ESP uses packages to deliver client libraries, frameworks, plugins and skeletons. ESP has a wide variety of extension packages to make your development more productive.

The Pak Manager

ESP uses the amazing Pak package manager and the online Pak Catalog of packages. The Pak utility is used to install, manage, upgrade and uninstall packages for ESP.

For example, to install Bootstrap using Pak:

$ pak install bootstrap
   [Install] jquery 2.1.3
   [Install] bootstrap 3.3.4

Behinds the scenes, Pak has downloaded Bootstrap into your local Pak cache — typically ~/.paks, installed Bootstrap into the local paks directory, and added bootstrap to the list of dependencies in the package.json. It also downloaded and installed the jQuery required dependency.

ESP Packages

ESP publishes many packages to support creating ESP applications. ESP skeletons are packages that include page layouts, navigation, stylesheets, fonts, other content and dependent packages.

Package NameDescription
esp-serverBasic server-side ESP support for serving ESP pages (only).
esp-skeletonESP application starter skeleton. Supports controllers and ESP pages.
esp-mvcESP MVC support. Includes scaffold templates to generate controllers, migrations, scaffolds.
esp-html-skeletonESP application starter skeleton for HTML5 MVC applications.
esp-angularESP support for AngularJS.
esp-angular-skeletonESP application starter skeleton for client-side AngularJS Applications.
